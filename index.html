Configuracion para Modo Sin Internet

Configure la red local para emergencias cuando no haya Internet

1. Router Local (Sin Internet)

Configure un router WiFi local. Las pantallas y la laptop de la directora deben conectarse a esta red.

SSID: ESCUELA-EMERGENCIA
Password: AlertaSegura2024
IP Router: 192.168.1.1

2. Laptop de la Directora

La laptop debe tener la IP estatica para ser el servidor de senalizacion:

IP: 192.168.1.100
Mascara: 255.255.255.0
Gateway: 192.168.1.1

3. IP de las Pantallas

Las pantallas obtendran IP automaticamente (DHCP) o use estaticas:

Rango: 192.168.1.101 - 192.168.1.160
Mascara: 255.255.255.0
DNS: 192.168.1.1

4. Iniciar en Modo Offline

En la laptop de la directora, abra el navegador y seleccione "Directora". En las pantallas, seleccione "Pantalla".

URL Director: file:///ruta/al/archivo.html
O abra el archivo directamente
El sistema detectara la red local automaticamente

5. Prueba de Conectividad

Verifique que todas las pantallas pueden "ver" a la directora:

1. Conecte todas las pantallas al WiFi
2. En la directora, verifique "Pantallas: 60/60"
3. Si no aparecen, refresque la pagina
4. El sistema usa WebRTC para conexion directa

6. Fallback Automatico

Si todo falla, las pantallas muestran mensajes pre-cargados:

Mensajes almacenados localmente:
- ALERTA ROJA: Lockdown
- ALERTA NARANJA: Terremoto
- ALERTA AMARILLA: Evacuacion
- TODO DESPEJADO: Fin de emergencia
